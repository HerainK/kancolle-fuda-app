# 艦これ 出撃制限札管理アプリ

「艦隊これくしょん(艦これ)」のイベント海域における出撃制限札(通称:札)の割り当てを計画するためのプランニングツールです。

実際のゲーム内出撃を記録するものではなく、出撃前に「艦娘を箱(艦隊)に仕分ける」ことで割り当て計画を立てるためのツールです。アプリ上の割り当ては自由に編集・やり直しができます。

## 特徴

- 完全にブラウザ内で完結する静的サイト(サーバー不要、バックエンドなし)
- データはすべて利用者自身のブラウザの localStorage にのみ保存され、他の利用者とは共有されません
- 艦娘マスタ(約330隻)を同梱。改装による艦種変化(例: 千歳 水上機母艦→軽空母)にも対応
- ドラッグ&ドロップ / タップ選択の両方で艦娘を艦隊へ割り当て可能
- 札の不一致・艦隊の最大収容数を自動でチェック
- JSONでのデータのエクスポート/インポートに対応(機種変更・バックアップ用)

## 技術スタック

- Vite + React + TypeScript
- Tailwind CSS
- dnd-kit(ドラッグ&ドロップ)
- react-router-dom

## 開発

```sh
npm install
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
```

## 艦娘マスタデータについて

`scripts/generate_ship_master.py` は [kcwiki/kancolle-data](https://github.com/kcwiki/kancolle-data) の公開データから艦娘マスタ(`src/data/shipMaster.json`)を生成するスクリプトです。名称・艦種などのテキスト情報のみを扱い、公式イラスト等の画像アセットは一切含みません。新艦娘の追加・修正はアプリ内の「艦娘マスタ管理」画面から手動でも行えます。
